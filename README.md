### Prerequisites

- **Node.js** (v14+)
- **npm** or **yarn**

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/GokulOfStarDust/English-Department-PSGCAS.git
   cd English-Department-PSGCAS
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Build for production:
5. 
   ```bash
   npm run build
   # or
   yarn build
   ```
